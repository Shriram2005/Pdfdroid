Frequently Asked Questions

Find answers to common questions about PdfDroid's PDF tools.

General Questions

Is PdfDroid free to use?

Yes, all our PDF tools are completely free to use. We don't have any hidden charges or premium features.

Do I need to create an account?

No, you don't need to create an account to use our tools. Just visit our website and start using the tools right away.

Are my PDF files secure?

Yes, all PDF processing is done locally in your browser. Your files are never uploaded to our servers, ensuring complete privacy and security.

Using Our Tools

What PDF tools do you offer?

We offer various PDF tools including:

  • PDF Metadata Editor
  • PDF Text Extractor
  • PDF Encryption
  • PDF Merger

What's the maximum file size I can process?

Since all processing is done in your browser, the file size limit depends on your device's memory. Generally, files up to 100MB work well.

Which browsers are supported?

Our tools work best with modern browsers like Chrome, Firefox, Safari, and Edge. Make sure to use the latest version for optimal performance.

Troubleshooting

Why isn't my PDF loading?

This could be due to:

  • The file is corrupted
  • The PDF is password-protected
  • Your browser needs to be updated
  • Insufficient device memory

How can I report a bug?

If you encounter any issues, please contact us with details about the problem and steps to reproduce it.

Privacy & Security

Do you store my PDFs?

No, we don't store any of your PDFs. All processing is done locally in your browser, and files are never uploaded to our servers.

Is my data protected?

Yes, we take data protection seriously. We use HTTPS encryption and implement various security measures to protect your privacy.
